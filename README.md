# EconomyLand
A plugin which allows your server to create and manage lands

## Build
- git clone https://github.com/EconomyS/EconomyLand && cd EconomyLand
- mvn clean
- mvn package

## Commands
- `/land <pos1|pos2|buy|sell|give|here|whose|list>`
